# ruby-cli-seg-sieve

## Description
Given a range low, high,
print all primes in this range.

## Tech stack
- ruby 2.4

## Docker stack
- alpine:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
